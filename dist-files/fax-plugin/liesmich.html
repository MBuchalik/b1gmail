Fax-Plugin Liesmich

Vielen Dank für den Erwerb des Fax-Plugins für b1gMail! In diesem Paket finden Sie alle Dateien, die Sie zur Installation benötigen.
Im Folgenden finden Sie eine Kurzanleitung zur Installation. Eine ausführliche Installationsanleitung ist in der Dokumentation zum Plugin enthalten. Diese Liesmich-Datei dient nur als Anleitung zum Schnell-Einstieg.

Installation

Gehen Sie bitte wie folgt vor, falls Sie noch keine ältere Version des Fax-Plugins nutzen und eine Erst-Installation durchführen möchten.
  1. Laden Sie alle Dateien und Ordner im Ordner "upload" dieses Download-Archivs per FTP in den Ordner Ihrer b1gMail-Installation hoch.
  2. Loggen Sie sich in den Admin-Bereich Ihrer b1gMail-Installation ein.
  3. Öffnen Sie die Seite "Plugins" im gleichnamigen Bereich.
  4. Wählen Sie die Register-Karte "Installieren".
  5. Wählen Sie als "Plugin-Paket" die Datei "Fax_(Lizenznummer).bmplugin", die in diesem Download-Archiv enthalten ist, aus und klicken Sie auf den Button "Installieren".
  6. Prüfen Sie die angezeigten Hinweise zur Plugin-Herkunft und klicken Sie dann auf den Button "Installieren".
  7. Wechseln Sie in den Administrations-Bereich des Fax-Plugins, indem Sie links im Menü auf "Fax" klicken.
  8. Rufen Sie die Register-Karte "Gateways" auf und geben Sie die Zugangsdaten für das Gateway ein, das Sie nutzen möchten. Klicken Sie danach auf "Speichern".
    Hinweis: Wenn Sie das SMSKaufen-Gateway nutzen möchten, empfehlen wir die Wahl des Gateway-Eintrags "SMSKaufen (HTTP)".
  9. Rufen Sie die Register-Karte "Einstellungen" auf und wählen Sie unter "Voreinstellungen" als "Standard-Gateway" das Gateway aus, das Sie verwenden möchten.
  10. Aktivieren Sie die Option "Fax" in den Gruppen-Einstellungen für die Benutzer-Gruppen, denen Sie den Fax-Versand erlauben möchten.
Falls Sie das SMSKaufen-Gateway verwenden, befolgen Sie bitte noch die folgenden zusätzlichen Schritte:
  1. Loggen Sie sich in den Kunden-Bereich von SMSKaufen ein.
  2. Rufen Sie den Menü-Punkt "Fax-Dienste" » "Preise und Einrichtung" auf.
  3. Geben Sie als Fax-Push-Adresse die Adresse
    (URL zu Ihrer b1gMail-Installation)index.php/faxPluginStatusPush/
    ein, wobei Sie den URL Ihrer b1gMail-Installation einsetzen. Beispiel:
    http://www.domain.xy/mail/index.php/faxPluginStatusPush/
    Klicken Sie dann auf "Push ändern".
Falls Sie das Computron-GNetX-Gateway verwenden, befolgen Sie bitte noch die folgenden zusätzlichen Schritte:
  1. Loggen Sie sich in den Kunden-Bereich ("Email-to-Fax") von Computron GNetX ein.
  2. Rufen Sie den Menü-Punkt "Email Sendeberichte" auf.
  3. Geben Sie als E-Mail-Adresse (Feld "Email") mailfrom und als Art Automatisch auswertbar an und klicken Sie dann auf "Einstellungen übernehmen".

Update von alter Fax-Plugin-Version

Wenn Sie das Fax-Plugin bereits in einer älteren Version nutzen, gehen Sie wie folgt vor, um auf die aktuelle Version zu aktualisieren. Ihre Daten bleiben beim Update erhalten.
  1. Deaktivieren/löschen Sie die alte Version des Plugins während des gesamten Prozesses nicht. Sie wird automatisch überschrieben.
  2. Laden Sie alle Dateien und Ordner im Ordner "upload" dieses Download-Archivs per FTP in den Ordner Ihrer b1gMail-Installation hoch.
  3. Loggen Sie sich in den Admin-Bereich Ihrer b1gMail-Installation ein.
  4. Öffnen Sie die Seite "Plugins" im gleichnamigen Bereich.
  5. Wählen Sie die Register-Karte "Installieren".
  6. Wählen Sie als "Plugin-Paket" die Datei "FaxPlugin_(Lizenznummer).bmplugin", die in diesem Download-Archiv enthalten ist, aus und klicken Sie auf den Button "Installieren".
  7. Prüfen Sie die angezeigten Hinweise zur Plugin-Herkunft und klicken Sie dann auf den Button "Installieren".
