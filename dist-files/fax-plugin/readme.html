Fax-Plugin Readme

Thank you very much for purchasing the Fax plugin for b1gMail! In this package, you can find all files you need to install the plugin.
This file contains summarized instructions how to set up the fax plugin and how to update it from further versions to the current version. More detailed instructions are contained in the plugin's manual - this readme file serves as a quick start introduction only.

Installation

Please follow the the following instructions to perform a fresh installation.
  1. Upload all files and folders contained in the folder "upload" in the download archive to the root folder of your b1gMail installation (e.g. using FTP).
  2. Log-in to the ACP of your b1gMail installation.
  3. Navigate to the page "Plugins" in the homonymous section.
  4. Choose the tab "Install".
  5. As "Plugin package", choose the file "Fax_(license-number).bmplugin", contained in this download archive and click at the button "Install".
  6. Check the plugin origin information displayed on the following page and click at the button "Install".
  7. Bring up the administration control panel of the Fax plugin by clicking at "Fax" in the menu on the left handside.
  8. Open the tab "Gateways" and enter the login information for the gateway you wish to use. Then click at "Save".
    Note: If you would like to use the SMSKaufen gateway, we recommend to chose the Gateway entry "SMSKaufen (HTTP)".
  9. Open the tab "Preferences" and chose the gateway you would like to use at "Default gateway".
  10. Activate the option "Fax" in the user group settings for all groups you would like to allow to send faxes.
If you use the SMSKaufen gateway, please follow the following additional steps:
  1. Log-in to the customers area of SMSKaufen.
  2. Open the menu entry "Fax-Dienste" » "Preise und Einrichtung".
  3. Enter the fax push address
    (URL to your b1gMail installation)index.php/faxPluginStatusPush/
    For example:
    http://www.domain.xy/mail/index.php/faxPluginStatusPush/
    Then click at "Push ändern".
If you use the Computron GNetX gateway, please follow the following additional steps:
  1. Log-in to the customers area ("Email-to-Fax") of Computron GNetX.
  2. Open the menu entry "Email Sendeberichte".
  3. As e-mail address (field "Email"), enter mailfrom and as "Art", chose Automatisch auswertbar. Then click at "Einstellungen übernehmen".

Updating from a prior Fax plugin release

If you already use a release of the Fax plugin, please follow these instructions to update to the current version. No data will be lost during the update.
  1. Do not deactivate or uninstall the old version of the plugin during the whole process. It will be overwritten automatically.
  2. Upload all files and folders contained in the folder "upload" in the download archive to the root folder of your b1gMail installation (e.g. using FTP).
  3. Log-in to the ACP of your b1gMail installation.
  4. Navigate to the page "Plugins" in the homonymous section.
  5. Choose the tab "Install".
  6. As "Plugin package", choose the file "FaxPlugin_(license-number).bmplugin", contained in this download archive and click at the button "Install".
  7. Check the plugin origin information displayed on the following page and click at the button "Install".
