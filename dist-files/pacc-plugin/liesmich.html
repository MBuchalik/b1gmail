PremiumAccount-Plugin Liesmich

Vielen Dank für den Erwerb des PremiumAccount-Plugins für b1gMail! In diesem Paket finden Sie alle Dateien, die Sie zur Installation benötigen.
Im Folgenden finden Sie eine Kurzanleitung zur Installation. Eine ausführliche Installationsanleitung ist in der Dokumentation zum Plugin enthalten. Diese Liesmich-Datei dient nur als Anleitung zum Schnell-Einstieg.

Diese Version des PremiumAccount-Plugins erfordert mindestens b1gMail 7.4.0-Beta4. Diese Version ist mit älteren b1gMail-Versionen nicht kompatibel!

Installation

Gehen Sie bitte wie folgt vor, falls Sie noch keine ältere Version des PremiumAccount-Plugins nutzen und eine Erst-Installation durchführen möchten.
  1. Loggen Sie sich in den Admin-Bereich Ihrer b1gMail-Installation ein.
  2. Öffnen Sie die Seite "Plugins" im gleichnamigen Bereich.
  3. Wählen Sie die Register-Karte "Installieren".
  4. Wählen Sie als "Plugin-Paket" die Datei "PremiumAccount_(Lizenznummer).bmplugin", die in diesem Download-Archiv enthalten ist, aus und klicken Sie auf den Button "Installieren".
  5. Prüfen Sie die angezeigten Hinweise zur Plugin-Herkunft und klicken Sie dann auf den Button "Installieren".

Update von PremiumAccount-2.x-Version aus b1gMail7

Wenn Sie das PremiumAccount-Plugin bereits in einer älteren Version der 2.x-Serie für b1gMail7 nutzen, gehen Sie wie folgt vor, um auf die aktuelle Version zu aktualisieren. Ihre Daten bleiben beim Update erhalten.
  1. Deaktivieren/löschen Sie die alte Version des Plugins während des gesammten Prozesses nicht. Sie wird automatisch überschrieben.
  2. Loggen Sie sich in den Admin-Bereich Ihrer b1gMail-Installation ein.
  3. Öffnen Sie die Seite "Plugins" im gleichnamigen Bereich.
  4. Wählen Sie die Register-Karte "Installieren".
  5. Wählen Sie als "Plugin-Paket" die Datei "PremiumAccount_(Lizenznummer).bmplugin", die in diesem Download-Archiv enthalten ist, aus und klicken Sie auf den Button "Installieren".
  6. Prüfen Sie die angezeigten Hinweise zur Plugin-Herkunft und klicken Sie dann auf den Button "Installieren".

Update von vorheriger PremiumAccount-Version aus b1gMail6

Wenn Sie von b1gMail6 auf b1gMail7 aktualisiert haben und in b1gMail6 das PremiumAccount-Modul (Version 1.77) eingesetzt haben, werden alle Daten bei der Installation des PremiumAccount-Plugins für b1gMail7 übernommen. Bitte prüfen Sie nach der Installation jedoch die Einstellungen und Ihre Rechnungs-Vorlage.
