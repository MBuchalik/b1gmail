PremiumAccount-Plugin Readme

Thank you very much for purchasing the PremiumAccount plugin for b1gMail! In this package, you can find all files you need to install the plugin.
This file contains summarized instructions how to set up the plugin and how to update from further versions to the current version. More detailed instructions are contained in the plugin's manual - this readme file serves as a quick start introduction only.

This PremiumAccount plugin version requires at least b1gMail-7.4.0-Beta4!

Installation

Please follow the the following instructions to perform a fresh installation.
  1. Log-in to the ACP of your b1gMail installation.
  2. Navigate to the page "Plugins" in the homonymous section.
  3. Choose the tab "Install".
  4. As "Plugin package", choose the file "PremiumAccount_(license-number).bmplugin", contained in this download archive and click at the button "Install".
  5. Check the plugin origin information displayed on the following page and click at the button "Install".

Updating from a prior PremiumAccount 2.x release for b1gMail7

If you already use a release of the PremiumAccount plugin from the 2.x branch, please follow these instructions to update to the current version. No data will be lost during the update.
  1. Do not deactivate or uninstall the old version of the plugin during the whole process. It will be overwritten automatically.
  2. Log-in to the ACP of your b1gMail installation.
  3. Navigate to the page "Plugins" in the homonymous section.
  4. Choose the tab "Install".
  5. As "Plugin package", choose the file "PremiumAccount_(license-number).bmplugin", contained in this download archive and click at the button "Install".
  6. Check the plugin origin information displayed on the following page and click at the button "Install".

Updating from a prior PremiumAccount release for b1gMail6

If you have updated your b1gMail6 installation to b1gMail7 and you used the PremiumAccount plugin (version 1.77) in b1gMail6, all data will be taken over to your new installation automaticaly. However, please check the plugin preferences and your invoice template after installing the plugin.
